# Popover 自定义提示

气泡提示信息


## Props
| Name      | Type | Default | Required | Description         |
| --------- | ---- | ------- | -------- | ------------------- |
| show      | Bool | false   | false    | 是否显示，例：true         |
| direction | Any  | 'right' | false    | 弹出方向，例：'left'       |
| title     | Any  |         | false    | 标题，例：'自定义跟随弹层'      |
| content   | Any  |         | false    | 内容，例: (<div></div>) |
