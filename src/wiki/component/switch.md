# Switch 开关

开关


## Props
| Name     | Type   | Default | Required | Description                 |
| -------- | ------ | ------- | -------- | --------------------------- |
| name     | String | ''      | false    | 组名，例：'group1'               |
| value    | Any    | ''      | false    | 值，例：1                       |
| checked  | Bool   | false   | false    | 是否选中，例：true                 |
| onChange | Func   | null    | false    | 点击回调，例：function(checked) {} |
