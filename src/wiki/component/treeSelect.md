# TreeSelect 树形选择器

树形选择器


## Props
| Name         | Type   | Default        | Required | Description |
| ------------ | ------ | -------------- | -------- | ----------- |
| data         | Array  | []             | false    | 树的相关数据      |
| showCheckBox | Bool   | true           | false    | 是否显示选择框     |
| marginLeft   | Number | 12             | false    | 每层缩进宽度      |
| onChange     | Func   | function () {} | false    | 被选中回调函数     |
