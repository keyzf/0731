# InlineEdit 点击编辑

点击产生输入框


## Props
| Name        | Type   | Default | Required | Description |
| ----------- | ------ | ------- | -------- | ----------- |
| value       | String |         | false    | 组件值         |
| onBlur      | Func   | null    | false    | 输入完成的回调     |
| placeholder | Any    | ""      | false    | 无值时显示的内容    |
